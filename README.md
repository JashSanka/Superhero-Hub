# SuperHero-Hub

SuperHero-Hub is a web project that aims to bring **all superhero movie information** from different universes into one place.

The goal is to create a clean, easy-to-navigate hub for exploring superhero movies, characters, and timelines from:
- **Marvel Cinematic Universe (MCU)**
- **DC Extended Universe (DCEU)**
- **Indian Superhero Universe**

This project is built using **HTML, CSS, and JavaScript only**.
